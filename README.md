# Virtual Calculator

This is a simple virtual calculator application created using Python's `tkinter` library. The calculator can perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features

- **Addition**
- **Subtraction**
- **Multiplication**
- **Division**
- **Clear Display**
- **Evaluate Expression**

## Requirements

- Python 3.x

## Installation

1. Ensure you have Python installed on your system. You can download it from [python.org](https://www.python.org/).
2. Clone this repository or download the source code.

```sh
git clone https://github.com/Sandeep-R-24/virtual-calculator.git
```

3. Navigate to the project directory.

```sh
cd virtual-calculator
```

## Usage

Run the calculator script using Python:

```sh
python calculator.py
```

This will open the calculator application window.

## How to Use

- Enter numbers and operators using the buttons provided.
- Press `=` to evaluate the expression.
- Press `C` to clear the display.

## Code Explanation

- The GUI is created using the `tkinter` library.
- The `click` function handles button clicks and updates the display.
- The `eval` function is used to evaluate the arithmetic expressions entered.

### Main Components

- **Display**: An entry widget to show the input and output.
- **Buttons**: Buttons for digits, operators, clear (C), and evaluate (=).

### Files

- `calculator.py`: The main script for the calculator.
